# Impact Rush — Game Design

## Vision

Impact Rush is a fast-paced mobile game designed for short, satisfying sessions in portrait orientation.

## Design Pillars

1. **Instant readability** — Players understand objectives within seconds.
2. **Responsive controls** — Input feels immediate on touch devices.
3. **Escalating tension** — Difficulty and stakes rise naturally over a session.
4. **Mobile-first pacing** — Sessions target 1–3 minutes per run.

## Target Platform

- **Primary:** Android phones (portrait)
- **Input:** Touch
- **Session length:** Short-form arcade

## Core Loop (Planned)

> Gameplay is not yet implemented. This section defines the intended loop.

1. Player enters from Main Menu
2. A run begins in the Gameplay scene
3. Player faces escalating challenges
4. Run ends with a score / result screen
5. Player returns to Main Menu or retries
